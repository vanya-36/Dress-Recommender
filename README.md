# Dress-Recommender
A simple dress recommender web app

### Running it locally

```bash
git clone https://github.com/vanya-36/Dress-Recommender.git
```

```bash
cd Dress-Recommender-master/dress_rec
```

```bash
export FLASK_APP=dress_rec.py
flask run
```


*  Go to 127.0.0.1/5000/ on your browser
